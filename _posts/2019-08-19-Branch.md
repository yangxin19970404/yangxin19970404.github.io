Github分支
============
# 分支
1. 分支创建  
    git   branch   分支名
2. 分支转换  
    git   checkout   分支名
3. 查看分支名
    git branch -v
4. 分支推进
    git   push   -u origin   分支名  
5. 分支合并  
    首先切换到要被合并的分支  
        git   merge   有新内容的分支名  
-----
    ## 切记，合并之后一定要推进提交
